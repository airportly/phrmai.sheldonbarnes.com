Tools on the road to HumanOS.

Interactive pharma-AI tools that communicate biology, drug discovery, and the 3D genome through direct manipulation — not slides. Each tool is self-contained; together they accrete into a unified HumanOS.

by Sheldon Barnes
Apps
Roadmap

Each tool is a self-contained folder in this repo that deploys as a static asset under phrmai.sheldonbarnes.com/<tool-name>/. A shared PHRMAI SDK (in progress) will extract the common primitives — 3D scaffolding, audio tours, click-to-learn info cards, imaging panels — so each new tool can focus on its own biology.

Planned: pathway browsers, single-cell explorers, drug-target interaction tours, narrated walkthroughs that connect molecular events to clinical outcomes.
